Set up your Claude Tracker

One small app on the computer where you use Claude Code keeps this tracker fed with your real usage. Install it once and forget it.

1 · Download the companion

The first time you open it, your OS may warn about an unidentified developer. macOS: right-click the app → Open. Windows: More info → Run anyway.

2 · Open it — that's it

  1. Double-click the downloaded app.
  2. It finds this tracker on your network by itself and sends your first reading.
  3. It sets itself to run at every login, so it keeps the tracker up to date from now on.

Needs Claude Code signed in on that computer — it reuses that login to read your real usage (it never signs in again). Nothing to copy, paste, or configure.

Prefer to run the script?

If you already have Python 3, grab the companion/ folder from the repo and run python3 companion.py — same auto-discovery, no download needed.
